1)Если делать Spring Boot с .jsp файлами через Intellij Idea, то нужно зайти:
Run -> Run/Debug Configurations -> Выбирать слева приложение -> Развернуть Environment -> В строке Working directory выбрать папку с проектом

Обязательно в application.properties дописать:
spring.mvc.view.prefix=/WEB-INF/views/
spring.mvc.view.suffix=.jsp

Создать структуру папок в main -> webapp -> WEB-INF -> views во views можно закидывать jsp

Обязательно добавить зависимость:
        <dependency>
            <groupId>org.apache.tomcat.embed</groupId>
            <artifactId>tomcat-embed-jasper</artifactId>
            <scope>provided</scope>
        </dependency>
